## TaskEcho：让会议待办自动同步到 GitHub Issues。

- 自动录音/转文字，通过 AI 提取会议摘要和待办事项，并一键创建或同步到 GitHub 仓库的 Issues 中。
- 极致轻量化，纯原生HTML实现，无后端和数据库！

### 2.运行

- 运行语音转文字.py
- 打开main.html

### 3.添加语音文件

- 将语音文件放到监控的文件夹下
- 过一会会自动在同目录下生成对应的文字
- 在页面选择文本，即可导入，并生成markdown形式的总结
- 确定提交内容和标题后，即可上传至issue！
